# Ploi default website template

This template is being used for newly created sites inside [ploi.io](https://ploi.io).

<p align="center"><a href="https://ploi.io" target="_blank"><img src="https://raw.githubusercontent.com/ploi-deploy/template/master/screenshot/screencapture-template-test-2021-07-24-14_40_10.png" width="100%"></a></p>


## Installing

Clone the repository

`npm i`

`npm run watch`

## Building

`npm run build`
